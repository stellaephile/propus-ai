Delhi Transit Intelligence Agent — CARES Statements
👩‍👧 Parents / Caregivers
Safety-first travel

As a parent traveling with my child, I struggle to find safe and nearby public transport stops, because many require walking through unsafe or congested streets, so I need reliable information on the closest and safest transit options, so that I can travel without worrying about my child’s safety.

Time-sensitive commuting

As a working parent, I struggle with unpredictable bus timings, because delays affect school drop-offs and work schedules, so I need accurate service frequency data, so that I can plan my day efficiently.

♿ Persons with Disabilities
Accessible transit gap

As a person with limited mobility, I struggle to access public transport, because stops are not within comfortable distance and lack accessible infrastructure, so I need information on nearby accessible transit options, so that I can travel independently.

Last-mile barriers

As a wheelchair user, I struggle with last-mile connectivity, because metro stations are not integrated with accessible bus stops, so I need insights on multimodal gaps, so that I can complete my journey without barriers.

👵 Elderly Users
Walking + heat stress

As an elderly person, I struggle to walk long distances to reach transit stops, especially in areas with low green cover and high heat, so I need access to nearby stops and shaded routes, so that I can travel comfortably.

Simplicity of information

As an older commuter, I struggle to understand complex transit systems, so I need simple, clear information about nearby stops and routes, so that I can use public transport confidently.

🎓 Students (School / College)
Affordable access

As a student, I rely on public transport but struggle when stops are far from my home or college, so I need equitable transit access, so that I can commute affordably.

Peak-hour congestion

As a college student, I struggle with overcrowded buses during peak hours, so I need better frequency and crowd insights, so that I can reach classes on time.

👩‍💼 Working Professionals
Commute inefficiency

As a working professional, I struggle with long commutes due to uneven transit access, so I need data-driven insights on better routes, so that I can save time.

Reliability gap

As a daily commuter, I struggle with unreliable transit services, so I need consistent frequency information, so that I can depend on public transport.

🏙️ Underserved Communities (CORE TO YOUR PROJECT)
Transit inequity

As a resident in a densely populated area with poor transit access, I struggle with limited mobility options, because planning does not consider population density, so I need equitable transit distribution, so that my area is not left behind.

Service neglect

As someone living in a peripheral ward, I struggle due to lack of frequent transit services, so I need visibility into underserved areas, so that authorities can improve services.

🌱 Environment + Urban Health
Urban stress exposure

As a city resident, I live in areas with high built-up density and low green cover, combined with poor transit access, so I need better planning insights, so that my neighborhood becomes more livable.

Sustainable mobility

As an environmentally conscious citizen, I struggle to choose sustainable transport options due to poor accessibility, so I need better-connected transit systems, so that I can reduce my carbon footprint.

🚶 Pedestrians / General Users
First-mile challenge

As a pedestrian, I struggle to reach transit stops safely due to poor walking infrastructure, so I need insights into accessible stops, so that I can use public transport easily.

Information gap

As a new user of public transport, I struggle to find nearby stops and routes, so I need intuitive and location-based information, so that I can navigate the system.

👩‍⚕️ Essential Workers
Off-peak mobility

As a healthcare worker, I struggle to find reliable transport during non-peak hours, so I need consistent service coverage, so that I can reach work on time.

🧑‍💼 Urban Planners / Policymakers
Data-driven planning

As an urban planner, I struggle to identify underserved areas, because traditional metrics ignore population and environmental stress, so I need integrated transit and RS insights, so that I can prioritize interventions effectively.

Resource allocation

As a policymaker, I struggle to allocate resources efficiently, so I need clear indicators of high-need areas, so that investments are targeted effectively.

🧍 Women Commuters
Safety + accessibility

As a woman commuter, I struggle to access safe transit options, especially in poorly connected areas, so I need information on safer and more accessible routes, so that I can travel confidently.
